List of movies API

A tiny Deno based REST API built for testing purposes.
The code is on GitHub.
Using TMDB to pull movie data.
Various experiments with this are on YouTube.

  • GET /init

    Get initial member data.
    Accepts: a X-MID request header to identify the user sending the request.
    Returns: a new unique user id if the X-MID header is not present.
    Returns: the movie lists associated with the user sending the request if the X-MID header is present.

  • GET /movies/search?key

    Search for movies by title.
    Required: A search key request parameter.
    Returns: a list of movie suggestions based on the key parameter.

  • POST /lists

    Create a movie list.
    Required: a X-MID request header to identify the user sending the request.
    Required: a Content-Type request header set to application/json.
    Required: the list name in the request body {"name": "My List"}.

  • DELETE /lists/:listId

    Delete a movie list.
    Required: a X-MID request header to identify the user sending the request.
    Required: the listId sent as a request path variable.

  • GET /lists/:listId

    Get a movie list.
    Required: a X-MID request header to identify the user sending the request.
    Required: the listId sent as a request path variable.

  • PUT /lists/:listId/movies/:movieId

    Add a movie to a list.
    Required: a X-MID request header to identify the user sending the request.
    Required: the listId sent as a request path variable.
    Required: the movieId sent as a request path variable.

  • DELETE /lists/:listId/movies/:movieId

    Remove a movie to a list.
    Required: a X-MID request header to identify the user sending the request.
    Required: the listId sent as a request path variable.
    Required: the movieId sent as a request path variable.
